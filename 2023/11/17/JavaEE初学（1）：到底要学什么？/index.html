JavaEE初学（1）：到底要学什么？

虽然课程名称叫JavaEE软件开发，但好像只讲了JSP相关技术，是否归结成JavaWeb学习更好？

简述

翻阅资料可知，JavaEE有十三种主要技术，JDBC、JNDI、EJB、RMI、Servlet、JSP、XML、JMS、Java IDL、JTS、JTA、JavaMail和JAF。（我不认识，不做展开，自行百度）

其次JavaEE开发通过框架使得开发变简单，如：

SSH: Structs + Spring + Hibernate

SSM: Spring +SpringMVC + MyBatis

学习路径

Servlet -> JSP -> Spring -> 组合框架

目前来说，我的前端知识也不完善，整体的这个技能树也不明晰，以黑马的视频（https://www.bilibili.com/video/BV1m84y1w7Tb）和某选修课的内容为主。后期再来总结@todo

CSS盒子模型

使用F12控制台后经常看到，原来出自这里

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。
